Conversion from Binary to Decimal

To convert any binary number B = bn bn-1 ... b1 b0 to decimal:

  1. Assign each binary digit (bi) a position, starting from 0 on the right (e.g., b0 is position 0).
  2. For each bit bi, calculate bi × 2i.
  3. Sum the results to get the decimal value.

Worked Example:

Let B = 1110, where b0 = 0, b1 = 1, b2 = 1, and b3 = 1.

Calculations:

  • b0 × 20 = 0 × 1 = 0
  • b1 × 21 = 1 × 2 = 2
  • b2 × 22 = 1 × 4 = 4
  • b3 × 23 = 1 × 8 = 8

Adding these: 0 + 2 + 4 + 8 = 14. Therefore, 1110 in binary is 14 in decimal.
