Assessment Overview

This section covers formative and summative assessments on the binary number system.

---

5.1 Formative Assessment

Group Discussion

Computers today use the binary system to represent information. It is called binary because only two different digits are used. It is also known as base two. Each zero or one is called a bit (binary digit).

Discussion Prompt: Why do you think computers use the binary system rather than decimal?

---

Worksheet 1: Binary Numbers

Students use cards representing the place value of a binary number to convert between binary and decimal numbers.

  • Encourage students to help each other or set a fixed time and review solutions together.
---

Worksheet 1 Answer Key

  • 5 = 00101
  • 3 = 00011
  • 12 = 01100
  • 19 = 10011
  • 8 = 01000
  • 15 = 01111
